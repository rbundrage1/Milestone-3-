Reggie Bundrage

I created American Football 101 as an educational website to help beginners understand the sport of football. Many people watch games without fully understanding the rules, scoring, positions, or strategy. This website was designed to make learning football easier in a simple and beginner-friendly format.

As someone who enjoys football, I wanted to build a project that combines my interest in sports with web development skills.

Website Purpose

The goal of this website is to teach the fundamentals of American football through organized pages and interactive content.

  • Learn key football concepts
  • Understand formations and positions
  • Study the history of the game
  • Review scoring and rules
  • Explore NFL teams
  • Test knowledge with a self-assessment quiz

Skills Demonstrated in This Project

This project was also used to practice and improve front-end web development skills.

  • HTML page structure and semantic layout
  • External CSS styling
  • Responsive web design using media queries
  • Navigation menu creation
  • Tables, images, and lists
  • JavaScript quiz functionality
  • GitHub Pages deployment
